# React - Infinite Scroll Example

## Steps to Setup

1. Clone this repository

```
git clone https://github.com/MarioCesarJr/react-infinite-scroll.git
```

2. Install dependencies

```bash
> yarn
```

3. Run app

```bash
> yarn start
```

Open <http://localhost:3000> to view it in the browser.
